# RuStoreApp_vk
Учебный проект RuStore - российский магазин приложений для Android

 Полная демонстрация работы приложения:
 [RuStore Demo Video] https://youtu.be/9v2h4aiAAQk

*Кликните на изображение для просмотра видео (2 минуты)*

Что показано в видео:
1. Запуск и онбординг
- Первый запуск приложения
- Экран онбординга с логотипом RuStore
- Приветственный текст и описание
- Кнопка "Переход на витрину приложений"

2. Главный экран - Витрина приложений
- Список всех приложений с карточками
- Иконка, название, разработчик, краткое описание для каждого приложения
- Категория и возрастной рейтинг
- Скроллинг списка

3. Фильтрация по категориям
- Нажатие на иконку категорий в правом верхнем углу
- Переход на экран со списком всех категорий
- Отображение количества приложений в каждой категории
- ильтрация списка приложений

4. Карточка приложения
- Нажатие на приложение "Вконтакте"
- Детальная информация о приложении:
  - Большая иконка
  - Полное название и разработчик
  - Категория и возрастной рейтинг
  - Горизонтальный список скриншотов
  - Полное описание приложения
  - Кнопка "Установить"

5. Просмотр скриншотов
- Нажатие на скриншот в карточке приложения
- Переход в полноэкранный режим просмотра
- Горизонтальный свайп между скриншотами
- Кнопка "Назад" для возврата

6. Демонстрация установки
- Нажатие кнопки "Установить" в карточке приложения
- Появление системного уведомления о загрузке
- Возврат к списку приложений

7. Навигация между экранами
- Использование кнопки "Назад" в шапке
- Возврат из карточки приложения
- Возврат из категорий
- Плавные переходы между экранами

Ключевые особенности:
Работа с данными:
- Локальная база данных приложений
- 6 тестовых приложений с полной информацией
- 5 категорий (Финансы, Инструменты, Игры, Государственные, Транспорт)
- Скриншоты и иконки для каждого приложения

Пользовательский интерфейс:
- Современный дизайн в стиле Material 3
- Адаптивная верстка под разные размеры экранов
- Плавные анимации переходов
- Удобная навигация

Функциональность установки:
- Работа с DownloadManager для загрузки APK
- Системные уведомления о прогрессе загрузки
- Интеграция с системным установщиком Android
- Обработка разрешений на установку

Архитектура приложения:
- Clean Architecture с разделением на слои
- MVVM паттерн с ViewModel
- Single Activity с Navigation Compose
- Репозиторий паттерн для работы с данными

### Технические детали реализации:
Тестовые данные:
Приложение использует локальные тестовые данные:
- СберБанк Онлайн (Финансы)
- ВКонтакте (Инструменты)
- Госуслуги (Государственные)
- Яндекс Метро (Транспорт)
- Шашки (Игры)
- Калькулятор++ (Инструменты)

собенности реализации установки:
В учебном проекте используются ссылки на страницы приложений в RuStore (например, `https://www.rustore.ru/catalog/app/...`). В реальном приложении эти ссылки были бы заменены на прямые URL для скачивания APK файлов. Тем не менее, реализована полная логика работы с DownloadManager, включая:
- Создание запросов на загрузку
- Уведомления о прогрессе
- Обработку завершения загрузки
- Интеграцию с системным установщиком

Архитектура API:
В проекте полностью реализована архитектура для работы с API:
- Retrofit клиент с настройкой интерсепторов
- Модели данных для API
- ApiRepository для обработки сетевых запросов
- Обработка ошибок и состояний загрузки
